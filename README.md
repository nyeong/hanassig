# 하나씩

배운 것들을 하나씩 정리합니다. 인덱스는 [notes/index.adoc](notes/index.adoc)을 참고해주세요.

- 라이센스: [CC BY-SA](LICENSE)

## 디렉토리 구조

```
├── assets/ -- 이미지 등 첨부파일
├── notes/  -- asciidoc 파일
│   └── index.adoc
│   └── ...
├── LICENSE
├── Gemfile
├── Gemfile.lock
└── README.md
```

## 글 구조

```asciidoc
= 글 제목
:description: 글 설명
:keywords: 키워드1, 키워드2
```

## 의존성

[asciidoc] 형식을 이용하고 있으므로 별다른 의존성 없이 텍스트 형태로 문서를
보실 수 있습니다.

[asciidoc]: https://asciidoc.org/
