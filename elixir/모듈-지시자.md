# 모듈 지시자

Module Directives.

## alias

모듈에 별칭을 붙일 때

```elixir
alias Foo.Bar, as: Bar

# Can now do
Bar.function
```

## import

다른 모듈의 함수와 매크로를 사용할 때

```elixir
import List, only: [duplicate: 2]

# Can now do
duplicate("hello", 3)
```

기본적으로 `import` 하지 않아도 `List` 모듈의 함수를 쓰는 데에는 지장이 없음. 다만 `List.duplicate`처럼 모듈을 명시해주어야함.

## require

모듈이 정의되고 컴파일되었음을 보장.

## use

해당 모듈의 콜백을 사용
