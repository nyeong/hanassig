# 리다이렉션

[Redirections (Bash Reference Manual)](https://www.gnu.org/software/bash/manual/html_node/Redirections.html)

프로그램의 입력과 출력을 다른 곳으로 돌리는 기능.

## 입력을 리다이렉션

```
$ command < input
```

## 출력을 리다이렉션

```
$ command > output
```

## 출력을 다른 곳에 덧붙이기

```
$ command >> output
```

## 출력과 에러를 모두 리다이렉션

```
$ command &> output
$ command >& output
$ command > output 2>&1
```
