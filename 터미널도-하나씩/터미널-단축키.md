# 터미널 단축키

- `<c-a>`: 행 앞으로 커서 이동
- `<c-e>`: 행 끝으로 커서 이동
- `<c-k>`: 커서부터 행 끝까지 내용 삭제
- `<m-f>`: 커서를 한 단어 앞으로 이동
- `<m-b>`: 커서를 한 단어 뒤로 이동
- `<m-d>`: 다음의 단어를 삭제
- `<m-delete>`: 이전의 단어를 삭제
- `<c-c>`: SIGINT
- `<c-z>`: SIGTSTP
- `<c-\>`: SIGQUIT
- `<c-d>`: EOT, EOF
- `<c-i>`: 탭

| `0x04` | EOT; end of transmission| `0x44` | D             |
| `0x08` | BS; backspace           | `0x48` | H             |
| `0x09` | HT; horizontal tab      | `0x49` | I             |
| `0x0A` | LF; line feed           | `0x4A` | J             |
| `0x0D` | CR; carriage return     | `0x4D` | M             |
| `0x1B` | ESC; escape             | `0x5B` | [             |
