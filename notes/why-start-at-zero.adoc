\n

\n== 본문

\n

\n자연수의 부분순열 2, 3, ..., 12를 줄임표 없이 표현하자면 아래와 같을 것이다:

\n

\n. stem:[2 <= i < 13]

\n. stem:[1 < i <= 13]

\n. stem:[2 <= i <= 12]

\n. stem:[1 < i < 13]
