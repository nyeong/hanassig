* 켜고 끄기

* 프로젝트 관리

하나의 emacs 화면에서 여러 프로젝트를 켜고 왔다갔다 할 수 있다.

| 키        | 설명                                   |
|-----------+----------------------------------------|
| ~SPC p a~ | 프로젝트 추가하기                      |
| ~SPC p p~ | 프로젝트 전환                          |
| ~SPC .~   | 현재 디렉토리에서 파일 찾기            |
| ~SPC f F~ | 현재 디렉토리에서 재귀적으로 파일 찾기 |
| ~SPC SPC~ | 프로젝트 내에서 파일 찾기              |

* 파일

| 키        | 설명                                   |
|-----------+----------------------------------------|
| ~SPC SPC~ | 프로젝트에서 파일 찾기                 |
| ~SPC .~   | 현재 디렉토리에서 파일 찾기            |
| ~SPC f F~ | 현재 디렉토리에서 재귀적으로 파일 찾기 |

|---------+------------------------------------|

| ~SPC <~ | 다른 버퍼로 바꾸기 (프로젝트 무관) |
